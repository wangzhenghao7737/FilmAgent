# 玩具总动员   

小男孩安迪的玩具们在他不在时会活过来。牛仔警长胡迪因新玩具巴斯光年的到来感到地位受威胁，两人从敌对到合作，共同踏上回家冒险。影片是首部全电脑动画长片，角色生动，情感真挚，开创皮克斯辉煌时代。 

# 千与千寻   

十岁女孩千寻随父母误入神灵世界，父母因贪吃变成猪。她被迫在汤屋工作，结识白龙、锅炉爷爷等伙伴，逐步找回勇气与名字。宫崎骏通过奇幻旅程探讨成长、贪婪与自我救赎，画面唯美寓意深刻。

 # 海底总动员   

小丑鱼马林为寻找被潜水员抓走的儿子尼莫，穿越广阔海洋。途中结识健忘的多莉，经历鲨鱼、水母群与悉尼港的重重险阻。影片色彩绚丽，父子情深动人，同时呼吁保护海洋生态。 

# 蜘蛛侠：平行宇宙  

 普通高中生迈尔斯·莫拉莱斯被放射性蜘蛛咬伤后获得超能力，并遇见来自其他宇宙的多位蜘蛛侠。他们联手阻止反派撕裂多元宇宙。影片采用漫画分镜与动态模糊技术，视觉风格革命性，故事热血又感人。 

# 飞屋环游记   

老人卡尔为实现亡妻梦想，用气球将房子升空飞往南美瀑布。途中意外带上童子军小罗素，两人与会说话的狗、稀有巨鸟展开冒险。影片前十分钟无对白叙事堪称经典，温情与幽默并存。
