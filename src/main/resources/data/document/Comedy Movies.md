# 布达佩斯大饭店   

讲述传奇酒店门童零与古斯塔夫先生卷入一场围绕贵妇遗嘱和名画的争夺战。影片以对称构图、高饱和色调和荒诞幽默展现欧洲旧时代的优雅与衰败，韦斯·安德森的独特风格贯穿始终，笑点密集又充满人文情怀。 

# 伴娘   

安妮因经济拮据和感情不顺陷入低谷，却仍担任闺蜜婚礼的首席伴娘。她与富有的新伴娘海伦明争暗斗，在筹备过程中闹出连串笑话。影片打破女性喜剧刻板印象，兼具爆笑桥段与真挚友情刻画。 

# 超级坏   

两位性格迥异的高中死党埃文与塞斯即将毕业，为参加心仪女生的派对，他们想方设法搞到酒。途中遭遇警察、怪邻居和尴尬误会，笑料百出。影片以青少年视角呈现成长焦虑，台词犀利又真实。 

# 帕丁顿熊2   

善良的帕丁顿熊为给露西婶婶买生日礼物，努力打工赚钱，却意外被陷害入狱。他在狱中结交朋友，最终洗清冤屈。影片充满英式幽默与温情，传递包容、礼貌与善意的价值观，适合全年龄段观众。 

# 摇滚校园  

 落魄摇滚乐手杜威冒充好友应聘小学代课老师，意外发现学生们极具音乐天赋。他组建儿童摇滚乐队，准备参加比赛。影片融合音乐与教育主题，充满活力与反叛精神，鼓励孩子勇敢表达自我。
