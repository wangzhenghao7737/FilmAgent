# 肖申克的救赎   

银行家安迪被冤判谋杀妻子及其情人，终身监禁于肖申克监狱。他凭借智慧与毅力赢得尊重，暗中策划越狱。影片通过他与狱友瑞德的友谊，展现希望、自由与人性坚韧，被誉为影史最伟大电影之一。 

# 阿甘正传   

智商只有75的阿甘从小受歧视，却阴差阳错参与越战、乒乓外交、捕虾创业等重大历史事件。他始终坚守善良与承诺，最终收获爱情与平静。影片以个人命运折射美国三十年变迁，温暖而励志。 

# 泰坦尼克号   

穷画家杰克赢得船票登上泰坦尼克号，与贵族少女露丝相爱。巨轮撞冰山沉没，杰克为救露丝牺牲。影片融合灾难、爱情与阶级冲突，视觉震撼，情感浓烈，成为全球文化现象。 

# 寄生虫   

韩国底层家庭金氏一家通过伪造身份，陆续潜入富豪朴社长家工作，表面和谐下暗藏危机。一场暴雨揭开两个家庭的阶级鸿沟与生存困境。影片黑色幽默与惊悚交织，获戛纳金棕榈与奥斯卡最佳影片。

 # 触不可及   

瘫痪的富豪菲利普雇佣刚出狱的黑人青年德里斯做护工。两人背景悬殊却建立起超越阶级的深厚友谊。影片改编自真实故事，幽默中饱含人性温度，探讨尊严、陪伴与生命的意义。
